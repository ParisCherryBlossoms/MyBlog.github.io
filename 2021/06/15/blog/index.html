Hexo+Kaze 搭建静态博客网站

2021-06-15 blog搭建 704 字
hexo kaze

静态博客网站搭建笔记

静态博客网站手册使用技术栈：

Git
Node.js
Hexo
Kaze

搭建流程

安装git

Windows：下载并安装 git
环境配置：
1.账号申请：gitee||github
2.配置全局帐号：

# 配置全局用户名
$ git config --global user.name "username"    //("username" 自己的账户名)

# 配置全局邮箱
$ git config --global user.email "username@email.com"    //("username@email.com" 注册账号时用的邮箱)

3.生成ssh

# gitee_ssh
$ ssh-keygen -t rsa -C "username@email.com" -f ~/.ssh/id_rsa_gitee

# github_ssh
$ ssh-keygen -t rsa -C "username@email.com" -f ~/.ssh/id_rsa_github

4.针对不同的域名使用不同的密钥对，配置known_hosts文件

# gitee_ssh
$ ssh-add ~/.ssh/id_rsa_gitee

# github_ssh
$ ssh-add ~/.ssh/id_rsa_github

5.测试

# gitee_ssh
$ ssh -T git@gitee.com

# github_ssh
$ ssh -T git@github.com

6.配置当前帐号：

# 配置当前用户名
$ git config user.name "username"    //("username" 自己的账户名)

# 配置当前邮箱
$ git config user.email "username@email.com"    //("username@email.com" 注册账号时用的邮箱)

安装node.js

Windows：下载并安装 node.js
版本要求：需不低于 10.13，建议使用 Node.js 12.0 及以上版本
环境配置：
∵安装模块默认路径为：【C:\Users\用户名\AppData\Roaming\npm】
∴
1.在安装目录中创建两个文件夹【node_global】及【node_cache】
2.打开cmd命令窗口，输入：

$ npm config set prefix "D:\software\nodejs\node_global"
$ npm config set cache "D:\software\nodejs\node_cache"

3.设置环境变量：
在【系统变量】下新建【NODE_PATH】，输入【D:\software\nodejs\node_global\node_modules】
将【用户变量】下的【Path】修改为【D:\software\nodejs\node_global】

安装hexo

$ npm install -g hexo-cli

初始化建站

1.选择文件夹
2.初始化

$ hexo init <folder>
$ cd <folder>
$ npm install

安装Kaze

进入theme文件夹，下载kaze模板：

$ cd themes
$ git clone https://github.com/theme-kaze/hexo-theme-Kaze.git

修改配置

修改站点配置文件 _config.yml

# Site
title:
subtitle:
description:
keywords:
author:
language: zh-CN
timezone:

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: hexo-theme-Kaze

新建文章

# [layout]：布局(post)
# <title>:标题&文件名称
$ hexo new [layout] <title>

网站托管

GitHub Pages 或 Gitee Pages

建立仓库

申请一个账号，创建一个新的仓库，仓库名尽量和账号名一致，这样可以避免一些因为路径引发的问题。

连接仓库

1.安装hexo-deployer-git

$ npm install hexo-deployer-git --save

2.修改站点配置文件 _config.yml

# URL
## Set your site url here. For example, if you use GitHub Page, set url as 'https://username.github.io/project'
url: https://parischerryblossoms.github.io
permalink: :year/:month/:day/:title/
permalink_defaults:
pretty_urls:
  trailing_index: true # Set to false to remove trailing 'index.html' from permalinks
  trailing_html: true # Set to false to remove trailing '.html' from permalinks

# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: https://github.com/ParisCherryBlossoms/ParisCherryBlossoms.github.io.git
  branch: master

上传仓库

$ hexo clean
$ hexo generate
$ hexo deploy

创建文章

$ hexo new "blog name"
