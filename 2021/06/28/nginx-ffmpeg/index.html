nginx+ffmpeg 搭建流媒体服务器（ubuntu）

2021-06-28 流媒体 277 字
nginx ffmpeg

静态博客网站手册使用技术栈：

nginx
ffmpeg
nginx-rtmp-module

搭建流程

安装环境

# gcc
$ sudo apt-get install gcc

# prce
$ sudo apt-get install gawk
$ sudo apt-get install build-essential
$ wget https://ftp.pcre.org/pub/pcre/pcre-8.45.tar.gz
$ tar zxvf pcre-8.45.tar.gz
$ cd pcre-8.45/
$ ./configure 
$ make
$ sudo make install

# zlib
$ wget https://www.zlib.net/zlib-1.2.11.tar.gz
$ tar -xzvf zlib-1.2.11.tar.gz
$ cd zlib-1.2.11
$ ./configure 
$ make
$ sudo make install

# nginx-rtmp-module
$ sudo apt-get install git
$ git clone https://github.com/arut/nginx-rtmp-module.git

# nginx
$ sudo apt-get install openssl
$ sudo apt-get install libssl-dev
$ wget https://nginx.org/download/nginx-1.21.0.tar.gz
$ tar zxvf nginx-1.21.0.tar.gz
$ cd nginx-1.21.0/
$ ./configure --add-module=../nginx-rtmp-module --with-http_ssl_module --with-pcre=../pcre-8.45
$ make
$ sudo make install

# ffmpeg
$ sudo apt-get install ffmpeg

# vlc
$ sudo apt-get install vlc

配置nginx.conf

末尾添加

rtmp {
    server {
        listen 2021;
        publish_time_fix on;
        application live {
            live on;
            record all;
            record_path /tmp/av;
            record_max_size 100M;
            record_unique on;
            allow publish all;
            allow play all;
        }
    }
}

运行

$ sudo /usr/local/nginx/sbin/nginx
$ ffmpeg -re -i "video path" -vcodec copy -acodec copy -f flv "rtmp://localhost:2021/live/123"
$ ffplay "rtmp://localhost:2021/live/123"
